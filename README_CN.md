# Flutter 学习项目

## 今日学习内容目录

1. **Flutter 项目结构**
   - 了解 Flutter 项目的基本结构
   - 各个文件夹和文件的作用

2. **Flutter Widget 基础**
   - 介绍常用的基础 Widget：`Container`、`Column`、`Row`
   - 如何使用它们来布局和组织界面

3. **状态管理**
   - 使用 `setState()` 管理简单的 UI 状态
   - 实现一个基本的计数器功能

4. **文件结构化**
   - 将功能代码拆分为单独的文件，提高代码的可维护性

5. **Git 提交规范**
   - 提交更新到 GitHub
